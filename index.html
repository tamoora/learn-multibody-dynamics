    Learn Multibody Dynamics¶

    This website contains the companion materials for TU Delft’s Multibody Dynamics B course taught by Jason K. Moore. It is organized as an online book with interactive code woven into the text. Each page can be downloaded as a Jupyter Notebook or Python script.
